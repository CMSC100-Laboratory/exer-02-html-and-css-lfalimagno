Steps on how to create GitHub page
1. rename html file to index
2. push it to repository
3. find repository
4. make repository public so it may be accessed by others, go to end of setting where you can find change visibility set to public
5. then go to setting, find pages seen on the left then select what branch you want to create a page, then save, then reload until a link would appear on top
6. you can now access the page by the url
